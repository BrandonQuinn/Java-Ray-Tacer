## Features

- Supported volumes: spheres, infinite plane
- Multiple point lights
- Point lights (inverse square intensity w/ quadratic attenutation)
- Orthographic camera
- Multi-threaded rendering (CPU)
- FSAA (Full-screen anti-aliasing)
